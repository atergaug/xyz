How To Update a Fedora Linux System

This quick tutorial shows one way to update a Fedora Linux install.

I’ll be honest with you, I didn’t have much experience with Linux distributions. Trying Fedora was a different kind of experience for me.

The first thing I did after installing Fedora was to try and install some software. I opened the software center and found that the software center was ‘broken’. I couldn’t install any application from it.

Sometimes we just ignore the updates and keep troubleshooting the issue we face. No matter how big/small the issue is – to avoid them, you should keep your system up-to-date.

For some reason, I cannot reliably load up the software center, so I prefer to utilize the dnf package managing commands to easily update the system. There are various possible methods to update your Fedora Linux system (i.e. from the software center or system settings) however, I will only cover the command line method.

Looking for Fedora version upgrade?

Keep in mind that updating Fedora means installing the security patches, kernel updates and software updates. If you want to update from one version of Fedora to another, it is called version upgrade and that is beyond the scope of the procedures here.

Updating Fedora Using The Terminal

Simply launch the terminal and type in the following command to start updating (you should be prompted to verify the root password):

sudo dnf upgrade

dnf update vs dnf upgrade

You’ll find that there are two dnf commands available: dnf update and dnf upgrade. Both command do the same job and that is to install all the updates provided by Fedora. Then why there is dnf update and dnf upgrade and which one should you use? Well, dnf update is basically an alias to dnf upgrade. While dnf update may still work, the good practice is to use dnf upgrade because that is the real command.
